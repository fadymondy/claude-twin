# @claude-twin/extension

Manifest V3 Chrome extension for [claude-twin](../README.md). Connects to the local MCP server's WebSocket bridge on `127.0.0.1:9997` and ports the 17 platform content scripts from `orchestra/twin-bridge`.

Implementation lands in [#4](https://github.com/fadymondy/claude-twin/issues/4) and following.
