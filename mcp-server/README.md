# @claude-twin/mcp-server

Local MCP server for [claude-twin](../README.md). Hosts a WebSocket bridge on `127.0.0.1:9997` for the Chrome extension to connect to, and exposes the browser command bus as MCP tools (`twin_*`).

Implementation lands in [#3](https://github.com/fadymondy/claude-twin/issues/3) and following.
